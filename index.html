1. បង្កើត Folder ថ្មីមួយ (ឧទាហរណ៍៖ My website)

2. បង្កើត Folder តូចខាងក្នុង Folder ធំ ពីរទៀត ដែលមានឈ្មោះ images និង style

3. ចូលទៅបើកកម្មវិធី Visaul Studio រួចចុច New file នោះយើងនឹងបានផ្ទាំងសរសេរកូដដែលមាន html,head,body,footer

4. ចុច file បន្ទាប់មកចុច save ឬ save as បន្ទាប់មកទៀតជ្រើសរើស Folder ដែលបានបង្កើតខាងដើម ហើយដាក់ឈ្មោះឲ្យ file របស់យើង(example: index.html)

5. ចំពោះការរចនាវេបសាយ យើងត្រូវបង្កើត Folder ថ្មីមួយទៀត ដោយដាក់ឈ្មោះនិងមានកន្ទុយសម្គាល់(.css) (example: style.css)
